# Tracking Images

The AR demo tracks these image targets:

```text
public/tracking/gallery-entrance.svg
public/tracking/workshop-station.svg
public/tracking/help-desk.svg
```

Attach or print any of those images in the real environment where you want the app to report location details.

Notes:

- Keep the image flat and fully visible.
- Use good lighting.
- Avoid glare.
- Keep the printed size close to the configured width in `src/main.js`.
